title: "Everything about React State Management"
description: "A deep dive in the React State Management world, from the simplest native options to the most complex external libraries."
abstract: |
    State is the cornerstone of a React application. Initially, all is well, you just have to handle a few component-local states and one or two well places useState do the trick. But quickly, things get out of hand: you have to include global state, you want to store API responses, persist the data between sessions and your application becomes a huge bowl of spahgetti.
    Nowadays, there are plenty of tools available in the React ecosystem to help you handle your application state. The variety of solutions can even be dizzying: from React native solutions to traditional store managers, to more exotic tools (URL-based state, network calls cache...), it can be hard to find your way in this jungle! Let's review all these solutions together, we'll see together what are the pros and cons of each tool and in which context they are the most appropriate.
    At the end of the talk, you will know the various state management tools of React inside and out!
